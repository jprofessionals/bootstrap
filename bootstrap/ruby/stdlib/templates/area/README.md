# {{area_name}}

A MUD area by {{namespace}}.

## Getting Started

Define your rooms in `rooms/`, items in `items/`, NPCs in `npcs/`, and daemons in `daemons/`.

Customize `mud_aliases.rb` to define your own short class names.
Customize `mud_loader.rb` to add or rename directories.
Customize `mud_web.rb` to control what data your web templates can access.
Add custom routes with `web_routes` in `mud_web.rb`.

## Publishing

Use `@project publish <name>` in-game to make this area live.
